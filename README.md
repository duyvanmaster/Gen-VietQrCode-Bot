## Tutorial

### Instruct
> Install [node.js](https://nodejs.org/en/download/)

> Configure `.env` file Change `TOKEN=<YOUR BOT TOKEN>`, `CLIENT_ID=<YOUR BOT ID>` And `GUILD_ID=<YOUR DISCORD SERVER ID>`

Bot setup command
```sh
npm install
```
Start the bot
```sh
node .
```
